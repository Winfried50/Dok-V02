    Die Dokumente werden ausschließlich lokal gespeichert.
    
            
    Sie werden nicht  über das Internet in einer Cloud gespeichert.
    
            
    Zum Speichern wird eine verschlüsselte DB innerhalb des Browser genutzt.
    
            
    Zum Hinzufügen oder zum Download der Dokumente wird das Verzeichnis "Download" des Handy,
                 Tablet oder Laptop/PC genutzt.
    
        
          
        
    
            
            
    Die Dokumente können als Bild- oder PDF-Dateien vorliegen.
    
            
    Die Bilddateien können die Formate .jpg, .jpeg, .tif oder .png haben.
    
            
    Die Dateinamen sollten nach einem Prinzip aufgebaut sein, nach dem sie leicht filterbar sind.
    
            
    In der APP können 3 Filter frei gestaltet werden:
    
            
    Der 1. Filter kann die Dokumenten-Art, der 2. Filter kann den Namen und der 3.Filter kann das Dateiformat beinhalten.

    Bei der Installation der App wird eine Basisvariante der Filter bereitgestellt.
                 Sie entspricht zum Teil den zuvor beschriebenen Beispielen.
            
    
            
    Diese Basisvariante sollte direkt nach der Installation bearbeitet,
                 d.h. persönlich angepasst werden.
    
            
    Anschließend sollte ein Backup der Filterparameter erfolgen.
                 Damit wird eine Sicherungsdatei der Filterparameter erstellt
                  und zugleich werden die Filterparameter im Browser gespeichert.
         
        
    
        
    
            
            
    Es werden alle vorhandenen Dokumente angezeigt.
                 Über die Filterauswahl kann die Anzahl der angezeigten Dokumente eingeschränkt werden. 
                 Mit einem Klick auf den Namen des Dokumentes wird das Dokument angezeigt.
                 Mit Klick auf "Download" wird das Dokument als einzelne Datei gespeichert und könnte gesendet oder gedruckt werden.
                
    
            
    
            
    Wenn ein Dokument ersetzt bzw. gelöscht werden soll,
                 dann muss zuvor "Ersertzen/Löschen" auf "Ja" gesetzt werden. 
         
        
    
        
    
            
            
    In diesem Menüpunkt können ein bzw. mehrere Dokumente hinzugefügt werden.
                 Achte darauf, dass die Dateinamen der Dokumente deinem Filterprinzip entsprechen.
            
    
        
    
        
    
            
            
    In diesem Menüpunkt können die Dokumente bzw. die Filterparameter 
                 aus einer zuvor erstellten Backup-Datei eingelesen werden (Restore).
    
            
    Dies kann notwendig werden, wenn der Cach-Speicher des Browser gelöscht wurde.
    
            
    
            
    Eine Backup-Datei der Dokumente sollte erstellt werden,
                 wenn ein bzw. mehrere Dokumente hinzugefügt, ersetzt oder gelöscht wurden.
    
            
    Eine backup-Datei der Filterparameter sollte erstellt werden,
                 wenn ein bzw. mehrere Filter hinzugefügt, geändert oder gelöscht wurden.
       
        
    
        
    
            
            
    In diesem Menüpunkt können die Filterparameter bearbeitet werden.
    
            
    Zu jeder Filteroption gibt es 2 Zeilen: "Anzeigename" und "Filtertext".
    
            
    Der Text "Anzeigename" wird bei der Filterauswahl angezeigt.
    
            
    Der "Filtertext" soll den Suchtext innerhalb des Dateinamens beinhalten.
                 Ein Dokument wird nur angezeigt, wenn dieser Text ein Teil des Dateinamens ist.   
            
    
            
    Wenn alle Dokumente angezeigt werden sollen "alle anzeigen",
                 dann muss der Filtertext leer bleiben.
